Base package for [tkviews](https://github.com/eumis/tkviews) and [wxviews](https://github.com/eumis/wxviews)

## Installation

Install using pip:

`pip install pyviews`

To install dev version:

`pip install git+https://github.com/eumis/pyviews.git@dev`
